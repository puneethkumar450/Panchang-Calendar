# Panchang Calendar

Panchang Calendar is a traditional Indian calendar web app with Panchang details, holiday layers, regional calendar overlays, and language selection.

## Features

- Full calendar browsing for years `2026` through `2099`
- Panchang data such as:
  - Tithi
  - Nakshatra
  - Yoga
  - Karana
  - Rasi / zodiac
  - Rahu Kalam
  - Yamagandam / Ketu timing
  - Ekadashi
- Holiday layers:
  - National holidays
  - Public holidays
  - State holidays
- Regional calendar overlays:
  - Tamil
  - Telugu
  - Gujarati
  - Bengali
- Language selector for:
  - English
  - Hindi
  - Telugu
  - Tamil
  - Kannada
  - Gujarati
  - Bengali
  - Marathi
  - Assamese
  - Odia
- Traditional temple-inspired UI with month shrine artwork

## Notes

- The app is fully client-side and does not require a build step.
- The `2026` public holiday dataset includes entries added from Timeanddate.
- Later years use generated recurring holiday logic combined with built-in state and public holiday rules.
